# ⚖ LexAI — Multilingual RAG Legal Assistant

AI that reads your legal contracts and tells you 
what you are actually signing — in your own language.

## Features
- 📝 Plain language summary
- ⚠ Risk analysis with scores
- 📋 Key terms extraction
- ❓ Lawyer questions generator
- 🌐 Supports Telugu, Hindi, English and 50+ languages

## Tech Stack
- Groq Llama 3.1 (Free LLM)
- Custom RAG Pipeline
- CUAD Knowledge Base
- Python + Google Colab

## How to Run
1. Open Google Colab
2. Upload all files
3. Run cells in order
4. Upload your contract and analyze!

## Disclaimer
LexAI is not a substitute for professional legal advice.
